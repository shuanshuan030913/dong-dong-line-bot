# 咚咚醬的 line bot


## Install & Develope in local

1. open ngrok
```
ngrok http 3000
```
2. project start
```
yarn
yarn start
```
3. go to [Line Developers](https://developers.line.biz/console/channel/1656110986/messaging-api?status=success)
4. change  `Webhook URL` link to ngrok https link
